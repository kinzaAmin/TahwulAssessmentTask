# Assets Folder

This folder contains static assets for the project.

## Structure

- `images/` - Store image files (PNG, JPG, SVG, etc.)
- `icons/` - Store icon files

## Usage

Import images in your components like this:

```javascript
import profileImage from '../assets/images/profile.jpg'
import bellIcon from '../assets/icons/bell.svg'
```

Or reference them directly in JSX:

```jsx
<img src="/src/assets/images/profile.jpg" alt="Profile" />
```
